react-resux › Subscriber

Class: Subscriber

Subscribers provide a way to link models' effects/reducers, so that they get triggered by the same non-namespaced action type, on a leading, latest, or every action basis. That is, they provide the means for generating redux sagas employing takeLeading, takeLatest, or takeEvery effects.

▸ takeEvery(actionType: string, actionGenerators: any): Subscriber

Defined in react-resux/src/subscriber.ts:119

Adds a subscription, which will watch for the provided actionType. It will default to taking every action and calling the respective action generator, which is equivalent to the behaviour of effects declared in models. It uses a takeEvery effect under the hood.
